About this static review

This is a streamlined, public version of the full analysis tool, simplified to run as a web page (GitHub Pages) with no server or database behind it. A few things are intentionally narrower than the full tool:

  • It focuses on the top 100 relationships (the strongest, "high-signal" ones), which are a sample of the larger set. Other relationships appear in the lists for reference but are not clickable here.
  • Individual relationship charts show the full statewide picture. The county filter still narrows the lists and summary numbers, but the per-relationship charts are not recalculated by county. In the full tool that recalculation runs on the server, which a static page cannot do.
  • A couple of free-text search boxes and one advanced prediction panel are simplified, and the data is a point-in-time snapshot.

Everything else behaves the same as the full tool.
