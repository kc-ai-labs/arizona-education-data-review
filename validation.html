Methodology & Caveats

  • The page shows the latest validation run only; historical runs remain stored in `analytics.validation_runs` and related tables.
  • Coverage status reflects column-level source discovery. Row-level matches and mismatches only exist for `source_confirmed` fields in the selected validation sample.
  • Manual review status is an adjudication layer. It does not update `analytics.schools` or rewrite automated validation result tables.
  • Evidence links point to the source family reference or drilldown hint captured during validation. CSV exports preserve the exact latest-run rows shown above.
  • Low initial match rates are acceptable during the POC as long as attempts and failure reasons are preserved for review.
